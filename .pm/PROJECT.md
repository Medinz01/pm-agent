# pm-agent

> Generated by pm-agent on 2026-03-16

---

## Purpose
A local-first AI project management agent that reads your repo, documents it, and watches for changes to build context-rich prompts.

## Goals
- Enhance code documentation
- Improve team collaboration through clear project understanding
- Automate the creation of project documentation

## Stack
Python, Ollama, Git, Watchdog

## Architecture
The project consists of a CLI tool that indexes and analyzes the source code, builds context-rich prompts using an LLM, and watches for changes to update the documentation.
